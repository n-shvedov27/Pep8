Проект pep8 содержит реализацию PEP8 валидатора.
Программа обрабатывает входные данные и выдаёт список ошибок в коде,
нарушающих правила и соглашения PEP8.
Модуль для исполнения - Pep8.py
Тестовый модуль - Pep8Tests.py
В проекте используются стандартные библиотеки sys, os, getopt, unittest.
Модули, необходимые для сборки проекта:
    Const.py - модуль с константами
    Checker.py - основной модуль, где происходят проверки.
Опции, который можно указать в параметрах командной строки:
 -e указание кодировочной таблицы для входных файлов
 -o файл для вывода результата (Если параметр не задан,
     вывод осуществляется в stdout)
 --help вывод справки
Далее задаётся список файлов для проверки или ожидается ввод с stdin.
Для окончания ввода с stdin введите "quit".
